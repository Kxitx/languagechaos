Dothraki Dictionary

Warning: This project is highly work in progress. Most planned features, like parsing of inflected forms, are not working yet, and the things that are already there might have some bugs. If you find something that seems to be not working right, or if you just have some ideas on how to improve it/which features you would like this dictionary to have, feel free to contact me! Thanks to all the people from the The Languages of Ice anf Fire Community Discord server for being so patient with all my stupid questions, make sure to also take a look at the wiki if you wanna learn more about the Dothraki language!
Enter a dothraki or English word. Since this in an early version, Dothraki words have to be in the dictionary form, but I'm working on getting parsing of inflected words to work!
